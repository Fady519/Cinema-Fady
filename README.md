## ✨ Features
- **Dynamic Content:** Fetches and displays up-to-date movie data.
- **Modern UI/UX:** Clean, dark-themed responsive interface designed for all screen sizes.
- **Search & Filter:** Easily search for your favorite movies and explore categories.
- **Optimized Build:** Powered by Vite for fast development and blazing-fast loading times.

## 🛠️ Tech Stack
- **Frontend:** React.js, JavaScript (ES6+)
- **Build Tool:** Vite
- **Styling:** CSS3 / Modern Layouts (Flexbox & Grid)
